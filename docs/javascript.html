启动Modal演示

使用bootstrap-modal

通过javascript调用modal:

$('#myModal').modal(options)

Options

名称 类型 默认值 描述
backdrop boolean true 加载一个class为modal-backdrop的背景元素。如果指定为static的话，点击背景时不会关闭对话框。
keyboard boolean true 按下Esc键时关闭对话框。
show boolean true 初始化时显示对话框.

Markup结构

你可以不用写一行javascript代码就能使用modals，只需要在需要绑定的元素上设置data-toggle="modal"，并且设置data-target="#foo"或href="#foo"，其中foo为modal元素的id。这样的话只要点击绑定的元素，就会启动这个modal。

除此之外，如果想要在modal初始化时添加指定选项，也可以把这些选项设置为绑定元素或者modal元素的data属性即可。

<a class="btn" data-toggle="modal" href="#myModal" >Launch Modal</a>

<div class="modal" id="myModal">
  <div class="modal-header">
    <a class="close" data-dismiss="modal">×</a>
    <h3>Modal header</h3>
  </div>
  <div class="modal-body">
    <p>One fine body…</p>
  </div>
  <div class="modal-footer">
    <a href="#" class="btn">Close</a>
    <a href="#" class="btn btn-primary">Save changes</a>
  </div>
</div>
注意！ 如果您想给modal增加动画进出的效果的话，只需要在.modal元素上添加.fade class（效果见上面实例演示中的效果）并且包含bootstrap-transition.js.

Methods方法

.modal(options)

将内容变成一个modal。可传入参数options object.

$('#myModal').modal({
  keyboard: false
})

.modal('toggle')

手动切换modal状态。

$('#myModal').modal('toggle')

.modal('show')

手动显示modal。

$('#myModal').modal('show')

.modal('hide')

手动隐藏modal。

$('#myModal').modal('hide')

Events事件

Bootstrap的modal类公开了一些事件，可以用来在调用modal方法时触发。

事件名称 描述
show 这个事件在show方法调用时触发。
shown 这个事件当modal显示时触发（将会等待css动画完成）。
hide 这个事件在hide方法调用时触发。
hidden 这个事件在modal被隐藏后触发。（将会等待css动画完成）。 
$('#myModal').on('hidden', function () {
  // do something…
})

使用bootstrap-tab.js

通过javascript启用tab标签切换:

$('#myTab').tab('show')

Markup结构

你可以不用写任何javascript就可以激活任何tab或者pill导航的标签效果，只需要简单的在这个元素上设置属性data-toggle="tab"或者data-toggle="pill"即可。

<ul class="nav nav-tabs">
  <li><a href="#home" data-toggle="tab">Home</a></li>
  <li><a href="#profile" data-toggle="tab">Profile</a></li>
  <li><a href="#messages" data-toggle="tab">Messages</a></li>
  <li><a href="#settings" data-toggle="tab">Settings</a></li>
</ul>

Methods方法

$().tab

设置tab标签元素和内容容器，tab标签需要设置`data-target`或者`href`属性指向到对应dom中的内容容器。 

<ul class="nav nav-tabs">
  <li class="active"><a href="#home">Home</a></li>
  <li><a href="#profile">Profile</a></li>
  <li><a href="#messages">Messages</a></li>
  <li><a href="#settings">Settings</a></li>
</ul>

<div class="tab-content">
  <div class="tab-pane active" id="home">...</div>
  <div class="tab-pane" id="profile">...</div>
  <div class="tab-pane" id="messages">...</div>
  <div class="tab-pane" id="settings">...</div>
</div>

<script>
  $(function () {
    $('.tabs a:last').tab('show')
  })
</script>

Events事件

事件名称 描述
show 这个事件在标签显示时，但是在新标签显示前才触发。可以分别使用event.target和event.relatedTarget来获得当前激活的标签和上一个激活的标签（如果存在的话）。
shown 这个事件在标签显示后才触发。可以分别使用event.target和event.relatedTarget来获得当前激活的标签和上一个激活的标签（如果存在的话）。 
$('a[data-toggle="tab"]').on('shown', function (e) {
  e.target // activated tab
  e.relatedTarget // previous tab
})

使用 bootstrap-tooltip.js

通过javascript来触发tooltip:

$('#example').tooltip(options)

Options

名称 类型 默认值 描述
animation boolean true 给tooltip增加一个css淡入淡出的动画效果
placement string|function 'top' 设置tooltip的位置 - top | bottom | left | right
selector string false 如果有设置selector选择器的话, tooltip对象将被委托给选择器指定的这个目标。
title string | function '' 如果`title`不存在的话，默认的title值
trigger string 'hover' 设置tooltip的触发方式 - hover | focus | manual
delay number | object 0

延迟显示或隐藏tooltip (ms)

如果设置为number，对于显示和隐藏都有指定的延迟效果

Object结构为: delay: { show: 500, hide: 100 }
注意！ 每个tooltip的选项也都可以通过data属性来设置。

Markup结构

由于性能问题，Tooltip和Popover的data-apis被设置为可选的。如果你想使用他们，可以初始化时传入selector选项。

<a href="#" rel="tooltip" title="first tooltip">hover over me</a>

Methods方法

$().tooltip(options)

绑定一个tooltip到一个元素集上。

.tooltip('show')

显示一个元素的tooltip。

$('#element').tooltip('show')

.tooltip('hide')

隐藏一个元素的tooltip。

$('#element').tooltip('hide')

.tooltip('toggle')

切换一个元素的tooltip。

$('#element').tooltip('toggle')

Using bootstrap-alert.js

Enable dismissal of an alert via javascript:

$(".alert").alert()

Markup结构

Just add data-dismiss="alert" to your close button to automatically give an alert close functionality.

<a class="close" data-dismiss="alert" href="#">&times;</a>

Methods方法

$().alert()

Wraps all alerts with close functionality. To have your alerts animate out when closed, make sure they have the .fade and .in class already applied to them.

.alert('close')

Closes an alert.

$(".alert").alert('close')

Events事件

Bootstrap's alert class exposes a few events for hooking into alert functionality.

Event Description

Options

Markup结构

Just add data-toggle="collapse" and a data-target to element to automatically assign control of a collapsible element. The data-target attribute accepts a css selector to apply the collapse to. Be sure to add the class collapse to the collapsible element. If you'd like it to default open, add the additional class in.

<button class="btn btn-danger" data-toggle="collapse" data-target="#demo">
  simple collapsible

<div id="demo" class="collapse in"> … </div>
Heads up! To add accordion-like group management to a collapsible control, add the data attribute data-parent="#selector". Refer to the demo to see this in action.
